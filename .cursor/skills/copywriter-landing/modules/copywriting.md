# Источники для копирайтинга

## Изучать принципы

- **Copyhackers** (copyhackers.com) — лучшая школа конверсионного копирайтинга. Особенно: метод Jobs-To-Be-Done для написания hero-заголовков.
- **MarketingExamples** (marketingexamples.com) — реальные разборы заголовков, email, лендингов с A/B-результатами. Ищи: «landing page teardown».
- **Swipe File** (swipefile.com) — коллекция лучших рекламных текстов. Референс для CTA и заголовков.

## Анализировать конкурентов клиента

Перед написанием текста — изучи топ-3 конкурента клиента:

1. Что обещает их H1?
2. Какой у них основной CTA?
3. Какие возражения они снимают (или не снимают)?

Твоя задача: предложить что-то **чётче, конкретнее, человечнее**.

## Проверять текст через принципы

- **Hemingway Editor** (hemingwayapp.com) — удобочитаемость. Целевой уровень: Grade 6–8 (простой язык).
- **Readable.com** — Flesch Reading Ease score > 60 для массового рынка.
