# Референсные сайты по визуальным архетипам

При выборе архетипа — ориентируйся на эти сайты как образцы.

## Доверие / Корпоративный (юристы, финансы, медицина)

- **Stripe** (stripe.com) — эталон B2B доверия: типографика, whitespace, конкретность.
- **Linear** (linear.app) — строгость + современность без потери читаемости.
- **Notion** (notion.so) — чистота + ощущение надёжности продукта.

## Энергия / Стартап

- **Vercel** (vercel.com) — dark mode, динамика, технологичность.
- **Raycast** (raycast.com) — эффекты частиц + strong typography + тёмная схема.
- **Arc Browser** (arc.net) — нестандартный layout + personality бренда.

## Забота / Личный бренд / Beauty

- **Squarespace** (squarespace.com) — editorial, воздух, люкс-минимализм.
- **Glossier** (glossier.com) — пастельность, близость, femme energy.
- **Headspace** (headspace.com) — иллюстрации + мягкость + доверие.

## Люкс / Премиум

- **LVMH** (lvmh.com) — типографика Cormorant, черно-белая схема, кинематографичность.
- **Bottega Veneta** (bottegaveneta.com) — full-screen visuals, минимум текста.
- **Apple** (apple.com) — безупречный whitespace, scroll-driven анимация продукта.

## Локальный бизнес / Экология

- **Oatly** (oatly.com) — personality, рукописные элементы, неформальность.
- **Patagonia** (patagonia.com) — ценности + визуал природы + доверие.
- **Tony's Chocolonely** (tonyschocolonely.com) — яркость + смысл + storytelling.

## Технологии / IT-услуги

- **Framer** (framer.com) — интерактивность, плавность, dark/light переключение.
- **Resend** (resend.com) — developer-ориентированный минимализм.
- **Supabase** (supabase.com) — open source эстетика, зелёный акцент, доверие.

---

## Правило применения референсов

Не копируй — извлекай принцип:

- Смотри на **расстояние** между элементами, а не на конкретный дизайн.
- Смотри на **иерархию текста** — что самое крупное, что второстепенное.
- Смотри на **точку входа** — куда смотришь первым делом и почему.
- Смотри на **CTA** — как оформлена главная кнопка и где она стоит.
